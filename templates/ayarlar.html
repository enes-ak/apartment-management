Sistem Bilgileri ve Tasima Rehberi
Veriler Nerede?

Tum veriler (odemeler, giderler, daire bilgileri, ayarlar, loglar) apartman.db adli tek bir dosyada tutuluyor. Bu dosya uygulama klasorunun icinde. PC kapansa da, uygulama kapansa da veriler kaybolmaz.

Yedekleme

Duzenli olarak apartman.db dosyasini yedekleyin. USB'ye, buluta veya baska bir klasore kopyalamaniz yeterli. Bu dosya = tum verileriniz.

Baska Bilgisayara Tasima
  1. Tum apartman klasorunu USB veya bulut ile yeni bilgisayara kopyalayin
  2. Yeni bilgisayarda Miniconda veya Anaconda kurulu olmali
  3. Terminal (komut satiri) acin ve sirayla calistirin: 
    conda create -n apartman python=3.11 -y
conda activate apartman
cd apartman_klasorunun_yolu
pip install -r requirements.txt
python app.py
  4. Tarayicida http://127.0.0.1:5000 adresine gidin

Veri kaybi olmaz. apartman.db dosyasi klasorun icinde oldugu surece tum verileriniz korunur.

Uygulamayi Calistirma (Her Seferinde)
conda activate apartman
cd apartman_klasorunun_yolu
python app.py
Sorun Giderme
  • Sayfa acilmiyor: Terminalde python app.py calistigina emin olun
  • Paket hatasi: pip install -r requirements.txt tekrar calistirin
  • Veriler gozukmuyor: apartman.db dosyasinin uygulama klasorunde oldugundan emin olun
